Javascript WYSIWYG HTML control

This is an example of a small, standards compliant, Javascript WYSIWYG HTML control written by Paul James. It currently works in Internet Explorer 5.5+ and Mozilla Gecko 1.3+ based browsers. To see what is going on, view the source.

The source is available in a convienient archive from this page, it is licensed under the BSD License which can be found at the foot of this page.

The button images are taken from OpenOffice.org and are licensed under the LGPL.
